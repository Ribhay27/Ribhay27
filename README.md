I'm a Data Science graduate from Arizona State University (Computer Science track, Dean's List) who likes working across the full lifecycle of a data problem — from ingesting messy real-world data to shipping models people can actually use. As a Software Engineer Intern at Initialyze, I built ETL pipelines and REST APIs on Databricks. ### Featured Projects

**[CareMetrics (Work In Progress)](https://github.com/Ribhay27/CareMetrics)** — Clinical Performance & Readmission Risk Analytics Pipeline<br>
End-to-end hospital quality analytics platform built on public CMS and CDC data. Airflow orchestrates ingestion, dbt transforms it into a PostgreSQL warehouse, and XGBoost/LightGBM models predict 30-day readmission risk with SHAP-based explainability — served through a FastAPI backend and a Streamlit dashboard.<br>
`Airflow` `dbt` `PostgreSQL` `XGBoost` `LightGBM` `SHAP` `FastAPI` `Streamlit`


**[HireLens](https://github.com/Ribhay27/hirelens)** — Job Market Intelligence Platform<br>
Scrapes job postings, pipes them through a PostgreSQL data pipeline, extracts in-demand skills with NLP, and surfaces hiring trends across a 5-page Streamlit dashboard.<br>
`Python` `PostgreSQL` `NLP` `Streamlit`

**[SBA Loan Default Prediction](https://github.com/Ribhay27/SBA)**<br>
ML pipeline predicting SBA 7(a) loan default risk from 1.2M+ historical loan records, surfacing which features matter most for repayment outcomes.<br>
`Python` `Machine Learning`

**[FitPro](https://github.com/Ribhay27/FitPro)**<br>
Full-stack fitness analytics app predicting calorie burn with Random Forest and XGBoost, with real-time tracking via PostgreSQL/Databricks and a Streamlit dashboard for progress visualization.<br>
`Python` `Random Forest` `XGBoost` `PostgreSQL` `Databricks` `Streamlit`

**[World Happiness Explorer](https://github.com/Ribhay27/World-Happiness-Report-Analysis)**<br>
End-to-end analysis of global happiness data — cleaning, EDA, and regression modeling to uncover the socio-economic drivers of national well-being, visualized in Tableau.<br>
`Python` `Tableau` `Regression`

**Real Estate Forecasting**<br>
Home price forecasting tool built with multiple regression analysis.<br>
`Python` `Regression`

**[EchoMind AI](https://github.com/Ribhay27/EchoMind)**<br>
AI journaling assistant that turns reflections into emotional summaries you can revisit over time, built with FastAPI and an LLM backend.<br>
`FastAPI` `LLM`
